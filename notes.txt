Creating a test express application with express-generator
- npm i -g express-generator
- express --view=ejs myapp
- cd myapp
- npm i
- commit
- npm i nodemon --save-dev
- npm i reload --save-dev
- add
  - var reload = require('reload'); // in myapp/bin/www
  - reload(server, app); // just after createServer
  - <script src="/reload/reload.js"></script> to myapp/views/index.ejs
- move error code from app.js to www just below reload()
- change package.json to "start": "cross-env DEBUG=myapp:* nodemon -e * ./bin/www"
  - run npm i -g cross-env (if it not installed yet)
- To see debug output you need to set the DEBUG environment variable
  - cmd: set DEBUG=myapp:*
  - powershell: $env:DEBUG="myap:*"
